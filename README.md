# Snake Game (JavaScript)

A simple browser-based Snake game built with plain HTML, CSS, and JavaScript.

## Features

- Grid-based snake movement with collision detection
- Food spawning and score tracking
- Best score saved in `localStorage`
- Pause/resume with Space
- Game Over overlay with Space to restart

## Controls

- Arrow Keys: Move the snake
- Space: Pause/Resume (or restart after Game Over)

## How to Run

1. Open `index.html` in any modern browser.
2. Use the arrow keys to start moving.

## Project Structure

- `index.html` - Game layout and scoreboard
- `style.css` - Board, snake, and UI styling
- `server.js` - Game logic and rendering

## Customization

- Change `gridSize` and `gameSpeed` in `server.js` to adjust difficulty.
- Tweak colors in `style.css` for different themes.
